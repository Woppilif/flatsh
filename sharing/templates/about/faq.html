Часто задаваемые вопросы

  • Предоставляемые услуги

В описании предоставляемых услуг используются следующие понятия:
Бот - программа, выполняющая обработку поступающих данных - запросов пользователя, предоставляющая возможности для учета рабочего времени. Проект - абстрактное понятие субъекта, над которым производится расчет отработанного времени. Сотрудник - лицо, имеющее доступ к определенному перечню проектов в определенном боте.

В пакете услуг LITE предоставляется возможность создать одного бота на любой из представленных платформ, создать один проект для 5 сотрудников.

В пакете услуг PRO предоставляется возможность создать одного бота на любой из представленных платформ, создать до трех проектов для 10 сотрудников. Устанавливать права доступа сотрудникам, на каждый из созданных проектов. Изменить тип авторизации работников на тип "По локации", что означает, что работники отправляют свои гео данные Telegram боту через функцию Трансляции местоположения для авторизации и начала работы над проектом.

При необходимости Вы можете увеличить количетсво ботов, проектов или сотрудников сверх пакета по отдельным тарифам: 1 бот - 300 рублей, 1 проект - 49 рублей, 1 сотрудник - 20 рублей.

Для начала работы с нашим сервисом необходимо на своем аккаунте иметь платную подписку по одному из представленных тарифов, создать Телеграм или VK бота, подключить бота к нашему сервису.

  • Выбор тарифного плана и пополнение баланса

На странице Тарифы представлены всевозможные тарифные планы сервиса. При необходимости увеличения количества сотрудников, ботов или проектов, при превышении лимитов в представленном тарифном плане, вы можете пройти процедуру создания одного из перечисленных видов услуг и произвести оплату по ним отдельно.

Для пополнения баланса необходимо перейти на страницу Пополнить баланс в разделе Аккаунт. Далее, произвести оплату на необходимую вам сумму. Средства зачислятся на ваш счет после успешного прохождения процедуры пополнения баланса и перехода по ссылке “Вернуться в магазин” на странице Яндекс.Касса.

  • Создание Телеграм бота

Для создания Телеграм Бота необходимо перейти в Телеграм Бот @BotFather и ввести команду /newbot.

 

После вам будет необходимо указать название вашего бота, которое будет отображаться как название чата для пользователей.

Далее необходимо указать короткий адрес бота, с помощью которого пользователи смогут начать работу, перейдя по ссылке. Короткий адрес бота должен содержать в себе слово bot, которое необходимо написать слитно с коротким адресом, либо через нижнее подчеркивание.

После успешного прохождения процедуры создания бота, @BotFather вышлет вам API Токен в формате 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11, который необходимо указать при создании бота в личном кабинете.

  • Создание VK бота

Для создания VK бота необходимо иметь или создать публичную страницу.

Перейдите в настройки публичной страницы и перейдите к пункту Работа с API. Во вкладке ключи доступа вы можете создать API ключ для взаимодействия вашего бота с нашими серверами. Нажмите на создать ключ и выберите пункт “Разрешить приложению доступ к сообщениям сообщества”.

После создания API ключа перейдите во вкладку Callback API. На этой странице перейдите во вкладку Типы событий и отметьте в пункте Сообщения Входящее Сообщение и Редактирование Сообщения.

После в правом меню выберите пункт “Сообщения” и перейдите к настройкам бота.

Параметры на этой странице должны соответствовать параметрам, представленным на скриншоте.

Далее, необходимо перейти во вкладку настройки сервера и установить у пункта “Версия API”: 5.80.

  • Подключение бота к серверам

Для подключения бота необходимо на странице Боты перейти по ссылке Создать бота.

В поле Токен необходимо указать токен, который был сгенерирован @BotFather или VK.

Для добавления VK бота необходимо указать строку подтверждения, сгенерированную VK (****** на скриншоте ).

А так же после подключения бота к нашим серверам необходимо вставить в строку “Адрес” URL, сгенерированный на странице создания бота (https://www.ewtm.ru/vk/<GUID> на скриншоте).

Важно! URL будет изменен после каждого сохранения VK  бота в нашем сервисе.

  • Создание проекта

Для создания проекта необходимо перейти во вкладку “Проекты” и выбрать пункт “Создать проект”. Далее заполните все представленные вам поля и выберите бота, к которому непосредственно будет относиться данный проект.

  • Добавление сотрудников и работа с сотрудниками

Сотрудники будут добавлены в ваш список автоматически после того, как они зарегистрируются в боте т.е после первого использования бота сотрудником. Изначально все сотрудники имеют Неактивный статус. Для того, чтобы сделать сотрудника активным и добавить его к проекту, он должен иметь статус Активен. Этот параметр (Пользователь активен) изменяется непосредственно на странице пользователя, так же как его оплата труда в час и коэффициент заработной платы.

  • Добавление сотрудника к проекту

Для того, чтобы сотрудник мог начать работать над проектом, его необходимо добавить в список работников, задействованных в этом проекте. Для этого перейдите на страницу Проекты, выберите необходимый проект. На странице проекта вы можете добавить сотрудника из списка всех имеющихся сотрудников. После нажатия клавиши “Добавить” сотрудник сможет начать работу над указанным проектом.
